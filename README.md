# CriminalIP Pyhton library


```
pip install criminalip
```

```
from criminalip.CriminalIP import IP
```

## Development
It requires `pipenv` to manage the requirements. And it also requires make command as optional
```
pipenv install
```

## Run Testing
```
pipenv run pytest
```
